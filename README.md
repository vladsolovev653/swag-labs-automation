# Swag Labs Automation
Проект с UI автотестами сайта https://www.saucedemo.com/ с моим видением Page Object Model, Page Elements и др.

Стек: node.js + TypeScript

## Начало работы
Установка зависимостей 

```bash
npm install
```

Запуск линтера 

```bash
npm run lint
```

Запуск тестов

```bash
npm run test
```
